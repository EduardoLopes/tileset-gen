Help

x

Let's say that each tile of your game it's 32x32 size. The size of the tileset base in this case had to be 96x64, and follow this pattern:

If you want 64x64 tiles, the base had to be 196x128. The math works like this:

32x32 tile:
width: 32 * 3 = 96;
height: 32 * 2 = 64;

48x48 tile:
width: 48 * 3 = 144;
height: 48 * 2 = 96;

64x64 tile:
width: 64 * 3 = 196;
height: 64 * 2 = 128;

You get the idea!

When upload the tileset base, you need to set the tile size in the app.

You can upload how much tilesets base you want, you can delete them if you want, and when all it's done, you can download a png of everything.
